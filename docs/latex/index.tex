\item Generic type object, the behavior is very similar to python.

\DoxyCodeLine{Storage B = A; \textcolor{comment}{// A and B share same memory, this is similar as python }}
\DoxyCodeLine{}
\DoxyCodeLine{Storage C = A.to(Device.cuda+0); }

\end{DoxyCode}

\end{DoxyItemize}\hypertarget{index_autotoc_md14}{}\doxysubsection{Tensor}\label{index_autotoc_md14}

\begin{DoxyItemize}
\item A tensor, API very similar to numpy and pytorch.
\item simple moving btwn CPU and GPU\+: 
\begin{DoxyCode}{0}
