\mbox{[}Note\mbox{]}
\begin{DoxyEnumerate}
\item The constructed \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}} will have same rank as the input \mbox{\hyperlink{classcytnx_1_1Tensor}{Tensor}}, with default labels, and a shared view (shared instance) of interal block as the input \mbox{\hyperlink{classcytnx_1_1Tensor}{Tensor}}.
\item The constructed \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}} is always untagged.
\item \mbox{[}Important\mbox{]} The internal block of \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}} is a referece of input \mbox{\hyperlink{classcytnx_1_1Tensor}{Tensor}}. That is, they share the same memory. All the change afterward on \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}} block will change in input \mbox{\hyperlink{classcytnx_1_1Tensor}{Tensor}} as well. Use \mbox{\hyperlink{classcytnx_1_1Tensor_a5e2248e9babdb786167ed349df9084ae}{Tensor.\+clone()}} if a shared view is not the case.
