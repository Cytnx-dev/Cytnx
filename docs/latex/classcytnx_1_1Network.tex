Format of a network file\+:


\begin{DoxyItemize}
\item each line defines a \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}}, that takes the format \textquotesingle{}\mbox{[}name\mbox{]} \+: \mbox{[}Labels\mbox{]}\textquotesingle{}
\item the name can be any alphabets A-\/Z, a-\/z
\item There are two reserved name\+: \textquotesingle{}TOUT\textquotesingle{} and \textquotesingle{}ORDER\textquotesingle{} (all capital)
\item One can use \textquotesingle{}TOUT\textquotesingle{} line to specify the output \mbox{\hyperlink{classcytnx_1_1UniTensor}{Uni\+Tensor}}\textquotesingle{}s bond order using labels
\item The \textquotesingle{}ORDER\textquotesingle{} line is used to specify the contraction order
\end{DoxyItemize}

About \mbox{[}Labels\mbox{]}\+:


\begin{DoxyItemize}
\item each label should seperate by a comma \char`\"{},\char`\"{}
\item one \textquotesingle{};\textquotesingle{} is needed and used to seperate Rowrank and column rank
\end{DoxyItemize}

About \mbox{[}ORDER\mbox{]}\+:


\begin{DoxyItemize}
\item The contraction order, it can be specify using the standard mathmetical bracket ruls.
\item Without specify this line, the default contraction order will be from the first line to the last line
