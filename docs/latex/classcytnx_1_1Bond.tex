\item the bond can have arbitrary multiple symmetries, with the type of each symmetry associate to the qnums are provided with the in\+\_\+syms.
\end{DoxyEnumerate}

\mbox{[}Note\mbox{]}
\begin{DoxyEnumerate}
\item if quantum number(s) are provided (which means the bond is with symmetry) then the bond MUST be tagged with either BD\+\_\+\+BRA or BD\+\_\+\+KET
